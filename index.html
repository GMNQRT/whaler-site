Frequently Asked Questions
How much does it cost ?

Nothing. Whaler is 100% free and does not require credit card information to start. It's an open source project and you can contribute to help it's development on our github repository.

Can I manage many Docker servers with only one instance of Whaler ?

Not at the moment. We currently working on this feature. We want that Whaler be able to switch between server directly inside the application. Stay tuned.

What do i need to make it works ?

You only need one computer with a running Docker server or getting access to a remote one. Thanks to Docker Whaler is totally cross platform, it will load and install all required functional dependencies.
